一夜限りの青春を作るために。

本イベントは、独自技術とMinecraftを用いたメタバース空間で作り上げる大規模プロジェクトです。
全員が安心して楽しむための「校則」をここに定めます。

⚠️ 適用範囲について

本規約は「生徒（参加者）」および「来場者」にのみ適用されます。
運営チーム（教員スタッフ等）は、演出・治安維持のためにこれらの制限を受けず、特権を行使する権利を有します。

🛑

第1章：絶対遵守事項 (Immediate BAN)

ここに記載された事項は「レッドカード」です。違反した場合、事前の通告なく即時BAN（永久追放）等の措置が執行される可能性があります。

1. 信用毀損・名誉毀損行為の完全禁止

本企画、運営、他参加者の社会的評価を下げる言動は、企画内外（SNS、裏垢、他鯖）を問わず処分の対象となります。

  • デマの流布、人格攻撃。
  • 「運営の不手際」や「不満」を過度に誇張して拡散するネガティブキャンペーン。

2. ハラスメント・迷惑行為

  • 差別・誹謗中傷: 人種、性別、信条への攻撃。
  • セクハラ: 執拗な接触、性的言動。
  • プラットフォーム規約違反: YouTube/Twitch等のToSに抵触するスキンや発言。（チャンネルBANリスクへの配慮）
  • ストーキング: 拒絶後の付きまとい、配信外での連絡強要。

3. 運営妨害・システム悪用

運営妨害

指示無視、暴言、リスナーへのファンネル行為（攻撃扇動）。

システム破壊

チートツール、ラグマシン、DDoS、バグの悪用。

⚖️

第2章：運営権限と制裁措置

本企画の秩序を守るため、運営は強力な権限を行使します。処分の判断は全て運営の裁量に委ねられます。

制裁レベルの定義

  • Timeout 一時的な行動制限。反省を促すための謹慎処分。
  • Event BAN 本企画（2026）への参加資格剥奪。期間中の再入場不可。
  • Permanent BAN 放らぼ主催の全イベント・コミュニティからの永久追放。

警告システム (/warn)

累積3回でシステムが自動的にBANを執行します。

🤝

第3章：プレイスタイル・マナー

🎬 ロールプレイの原則

  • 「生徒」になりきる: メタ発言は控えめに。世界観を大切に。
  • 初対面設定: 知り合いでも「クラスメイト」として接する。
  • 恋愛禁止: カップル成立等の恋愛RPは全面禁止です。清らかな交際を。

📹 配信・録画の義務 (常時REC)

自衛のため、プレイ中は原則として常に録画・配信を行ってください。トラブル時の証拠がない場合、救済できません。

🎭 際どいRPと事前合意

対立、裏切り、過度なイジり等は、必ず裏で事前に合意形成を行ってください。合意なき敵対は「荒らし」です。

🔥 エンジョイ精神

PvP等のイベントでは、勝利（キルムーブ）よりも「その場の盛り上がり」を優先してください。魅せるプレイを！

⚖️

第4章：特権利用に関する誓約

"Noblesse Oblige"（高貴なる者の義務）

コマンドや特殊Modなどの強大な力は、あなたの「良心」と「信頼」の上に成り立っています。

❌ Bad

私利私欲のために使う（金策短縮、PvPでの不正勝利）。

⭕️ Good

「誰かを楽しませる」「凄い演出を作る」ために使う。

※操作ミスで事故が起きたら、隠さずすぐに報告してください。正直な報告にはペナルティはありません。

📦 システム・著作権・視聴者対応

ベータ版システムについて

独自Modは開発中です。予期せぬバグ、データ巻き戻り、アイテム消失が発生する可能性があります。運営は補償義務を負いません。「形あるものはいつか壊れる」精神でお願いします。

著作権と配信許諾

配信・動画化・収益化は自由です。他視点への映り込みに同意したものとみなします。BGM（AVC/メディアプレイヤー含む）の権利処理は各自で行ってください。

Must Include

📺 視聴者へのマナー周知義務

以下のテンプレートを、配信の概要欄や固定コメントに必ず掲示してください。 

■ 放らぼ文化祭 視聴者ガイドライン
本企画を楽しむために、以下の「お・ひ・た・し」を守ってご視聴ください。
【お】お気持ちコメントをしない（批判・説教・自治厨NG）
【ひ】秘密をバラさない（他視点の情報を書く「鳩行為」NG）
【た】他人の関係を強要しない（「〇〇と付き合って」等のコメントNG）
【し】指示を出さない（求められていないアドバイス・指示厨NG）

ここは「学校」という舞台です。
映画やドラマを見るような気持ちで、生徒たちの物語を見守ってください。

📝 その他・例外条項

包括的禁止事項（常識的判断）

本規約に明記されていない事項であっても、「一般常識」や「社会通念」から著しく逸脱する行為や言動は禁止とします。 ルールの隙間を突くような行為も、運営判断で処分の対象となります。

規約の改定について

運営は、必要に応じて事前の予告なく本規約を変更・改定する権利を有します。 変更後の規約は、Discordでの告知をもって即時効力を発揮します。
